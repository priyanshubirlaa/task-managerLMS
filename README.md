# Task Manager Dashboard

A Vercel-friendly dashboard app built with Next.js and TypeScript.

## Features
- Dashboard data visualization
- Component-based UI composition
- Status and priority filtering
- Mock backend API route
- Deployable without any server-only services

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Use the default Next.js settings.
4. Deploy.

The app uses only Next.js and React, so it works well on Vercel's free plan.
